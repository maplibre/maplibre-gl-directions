    • Protected

      Creates route lines from the server response.

      Each route line is an array of legs, where each leg is an array of segments. A segment is a Feature|GeoJSON LineString Feature. Route legs are divided into segments by their congestion levels. If there's no congestions, each route leg consists of a single segment.
